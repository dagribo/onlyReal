Протяжка копий (локальная оптимизация)

Так как оптимизация локальная, то она протягивает копии в коде в рамках одного базового блока. Например:

Исходный код:         Оптимизированный:
a = x;                a = x;
b = a;                b = x;
c = a + 3;            c = x + 3;

Применение оптимизации:

var pc = new PullCopies(treeCode);   //Создание экземпляра класса оптимизации, по исходной программе treeCode
var new_code = pc.Optimize();        //Вызов метода, оптимизирующего код. Возвращает трехадресный код программы после оптимизации (протяжки копий)
