Устранение переходов через переходы + удаление пустых операторов (пример оптимизации можно посмотреть в презентации Opt5, слайд 32
Задача предполагает проверку по трёхадресному коду
Создать класс NonZero_JTJ, из его экземпляра вызывать функцию DeleteJumpThroughJump()

t1.DeleteJumpThroughJump();
