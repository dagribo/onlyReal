Устранение локальных общих подвыражений построением ациклического графа

Так как оптимизация локальная, то она заменяет общие подвыражения в коде в рамках одного базового блока. Например:

Исходный код:         Оптимизированный:
    a = b;            a = b;  
	c = b - a;        c = b - b;
	d = c + 1;        d = c + 1;
	e = d * a;        e = d * b;
	a = x - y;        a = x - y;
	k = c + a;        k = c + a;

Применение оптимизации:

var dag = new DAG(treeCode);   //Создание экземпляра класса оптимизации, по исходной программе treeCode
var new_code = dag.Optimize();        //Вызов метода, оптимизирующего код. Возвращает трехадресный код программы после оптимизации (устранение локальных общих подвыражений построением ациклического графа)
